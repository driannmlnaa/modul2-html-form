Jika Anda mengklik tombol "Kirim", data formulir akan dikirim ke halaman bernama "/action_page.php".

1. saat mengisi data dan tombol reset di-klik maka value form input akan kembali menjadi Budi dan Prawira karena telah menjadi default value.

2. saat tombol kirim di-klik maka data yang telah diisi akan dikirim ke halaman /action_page.php, namun dikarenakan file action_page.php tidak ada maka akan ada pesan "Cannot get action_page.php".

3. saat diberikan atribut target dan diisi _self akan menampilkan halaman yang sama seperti sebelumnya, namun jika _blank maka akan ditampilkan pada tab baru.

4. saat menambahkan method get url berubah menjadi: file:///G:/action_page.php?i_nama=Budi&i_belakan=Prawira
